# StudyForge

StudyForge is an AI-powered academic co-pilot for tasks, research, flashcards, notes, tutoring, study guides, and homework scanning.

## Project

The app lives in `CLAUDE/studyforge/`:

- `CLAUDE/studyforge/backend` - Express API and PostgreSQL schema
- `CLAUDE/studyforge/frontend` - React + Vite frontend
